## Introduce

This project simply is an application schedueling time table for personal or team project. Calendarize was built in Sping boot technology for backend and ReactJs for client UI

## Features

- Using microservices architecture built in Spring boot , services included: project-service to handle data and logic about personal tasks and team project's tasks
- Using Jpa Repository for handle database interaction
- Feign client and WebClient for services interaction
- Oauth2 (github provider) and basic auth for login
- Using eureka server for discovery server
- Using apigateway-server to customize paths, cors handling, authentication and authorization
- Using ReactJs to implement clients's UI
